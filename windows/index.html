Simple tutorial on how to install windows 10/11 onto a new computer

Step 1: Download a windows image, from microsoft or a custom pretweaked image, i recommend revision

Step 2: Download rufus

Step 3: Use rufus to load your windows image onto a usb stick (at least 8gb) using the default stock settings

Step 4: Install windows onto your computer and follow the instructions, be sure to select pro edition and to put this on your main drive

Step 5: Once inside of windows, use massgraves legal and free activation software and select hwid activation

Step 6: Your new computer has an activated, legal, copy of windows for free
